# Git Setup Commands for Azure Virtual Machine Azmon

# After installing Git, run these commands in your project directory:

# 1. Initialize the repository
git init

# 2. Add all files to staging
git add .

# 3. Create initial commit
git commit -m "Initial commit: Azure Virtual Machine Azmon project setup"

# 4. Add your GitHub repository as remote (replace YOUR_USERNAME with your GitHub username)
git remote add origin https://github.com/YOUR_USERNAME/azmon-ampls.git

# 5. Push to GitHub
git branch -M main
git push -u origin main

# Alternative: If you want a different repository name
git remote add origin https://github.com/YOUR_USERNAME/azure-vm-azmon.git
